# Introducción

Nekos API es una API gratis que provee tus proyectos de images de chicas gato (catgirls). La API es de codigo abierto y se puede encontrar GitHub.

## Copyright y licencias

El codigo fuente de la API es abierto y esta licenciado bajo la MIT license. Ten en cuenta que las imagenes proveidas por la API no son nuestras y los dueños de estas se guardan los derechos de las imagenes (por lo tanto, no te damos una licencia por las imagenes).

## Que puedes y no puedes hacer

**Puedes**:
- Tu _puedes_ usar la API para tus proyectos personales y comerciales.

**No puedes**:
- Tu no puedes almacenar los datos proveidos por la API.

*NOSOTROS PODEMOS, EN CUALQUIER MOMENTO Y SIN PREVIO AVISO, BLOQUEARTE EL ACCESO A LA API. ESTO LO HAREMOS EN CASO DE QUE CONSIDEREMOS QUE EL USO QUE HACE DE LA API ES INDEBIDO*

## Crediting

Although you don't need to credit the API, we would appreciate it if you did. You can credit the API by adding a link to the API's GitHub page in your project's description.

# Introduction

Nekos API is a free to use API that provides a variety of images and gifs of catgirl. The API is open source and can be found on GitHub.

## Copyright and licenses

The API's source code is open source and licensed under the MIT license. However, the images served by the API are from their respective owners and they are the copyright holders (and therefore we are not licensing the images to you).

## DOs and DONTs

**DOs**:
- You _can_ use the API in your commercial and non-commercial projects.

**DONTs**:
- You _must not_ store the content served by the API for more than an hour. This includes but is not limited to any API response, partially or complete.

*WE MAY, AT ANY TIME AND WITHOUT PRIOR ADVICE, BLOCK YOUR ACCESS TO THE API. WE WILL DO THIS IN CASE THAT WE CONSIDER THAT YOUR USE YOU MAKE OF THE API IS INADEQUATE*

## Crediting

Although you don't need to credit the API, we would appreciate it if you did. You can credit the API by adding a link to the API's GitHub page in your project's description.

# Introducción

Nekos API es una API gratis que provee una gran variedad de imagenes de anime. Es de codigo abierto (el cual puede ser encontrado en GitHub).

## Copyright y licencias

La API es de codigo abierto y esta licenciada con la MIT License. A pesar de esto, las imagenes proveidas por la API son de sus respectivos dueños y ellos son los titulares de los derechos de autor (y por consecuencia nosotros no podemos licenciarte las imagenes).

## Términos de uso

No tenemos ningun documento legal que podamos darte. Puedes encontrar nuestro reglamen [here](/docs/terms-of-use).

## Dar creditos a la API

Aunque no hace falta que nos des creditos, lo agradeceriamos si lo hicieses. Puedes dar creditos a la API añadiendo un link a este sitio web en la descripcion de tu proyecto o en otra parte.

The `nekosapi` Typescript library is a (community made) officially supported API warapper for Nekos API.

## Installation

The library can be installed using Yarn, PNPM or NPM:


<Tabs items={['Yarn', 'PNPM', 'NPM']}>
    <Tab>
        ```bash
        yarn add nekosapi
        ```
    </Tab>
    <Tab>
        ```bash
        pnpm install nekosapi
        ```
    </Tab>
    <Tab>
        ```bash
        npm install nekosapi
        ```
    </Tab>
</Tabs>

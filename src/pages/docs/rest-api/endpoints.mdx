# Endpoints

Base url: `https://nekos.nekidev.com/api`

## `GET /image`

Returns a random image from the specified category.

### Parameters

| Name | Type | Description | Required |
| :--- | :--- | :--- | :--- |
| `categories` | array of strings | The category to get the image from. | No |
| `limit` | number | The amount of images to get. (max: 25) | No |

### Response

```js
{ 
    'data': [
        {
            'id': 'xxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx',
            'url': 'https://example.com/assets/xxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx.png',
            'artist': "xxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
            'source': {
                'name': 'xxxxxxx',
                'url': 'https://example.com/xxxxxxx'
            },
            'nsfw': false,
            'categories': [
                'xxxxxxx',
                'xxxxxxx',
            ],
            'createdAt': '2021-01-01T00:00:00.000Z',
            'meta': {
                'eTag': 'xxxxxxxxxxxxxxxxxxxxx',
                'size': 6336807,
                'mimetype': 'image/png',
                'dimens': {
                    'width': 1024,
                    'height': 1024,
                },
            }
        }
    ],
    'success': true
}
```

### Example

<Tabs items={["JavaScript", "Python 3"]}>
    <Tab>
        Using the `fetch` API:
        ```js
        const res = await fetch('https://nekos.nekidev.com/api/image?categories=neko')
        const json = await res.json()
        console.log(json.images.url)
        ``` 
    </Tab>
    <Tab>
        Using the `requests` library:
        ```py
        import requests
        res = requests.get('https://nekos.nekidev.com/api/image?categories=neko')
        print(res.json()['images']['url'])
        ```

        Using the `anime-api` library:
        ```py
        from anime_api.apis import NekosAPI
        api = NekosAPI()
        image = api.get_random_image(categories=['neko'])
        print(image.url)
        ```
    </Tab>
</Tabs>

## `GET /artist/{id}/images`

Get all images by an artist.

### Parameters

| Name | Type | Description | Required |
| :--- | :--- | :--- | :--- |
| `id` | string | The ID of the artist to get images from. | Yes |
| `limit` | integer | The maximum number of images to get. | No |
| `offset` | integer | The number of images to skip. | No |

### Response

{

        {
            'id': 'xxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx',
            'url': 'https://example.com/assets/xxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx.png',
            'artist': "xxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
            'source': {
                'name': 'xxxxxxx',
                'url': 'https://example.com/xxxxxxx'

            'categories': [
                'xxxxxxx',
                'xxxxxxx',

            'meta': {
                'eTag': 'xxxxxxxxxxxxxxxxxxxxx',
                'size': 6336807,
                'mimetype': 'image/png',
                'dimens': {
                    'width': 1024,
                    'height': 1024,
                },
            }
        }
    ],
    'success': true,
}
